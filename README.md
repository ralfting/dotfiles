### Ralph's dotfile

## Commands

| Shortcut            |       Description        |
| ------------------- | :----------------------: |
| `<F3>`              |        Paste Mode        |
| `<leader>b`         |       History back       |
| `<leader>tf`        |  Run current file test   |
| `<leader>tl`        |      Run last test       |
| `<leader>b`         |      Open nerd-tree      |
| `<leader><leader>w` | Navigate in current file |
| `<space>gd`         |         Git diff         |
| `<space>gb`         |        Git blame         |
| `<space>gs`         |        Git status        |
| `<ctrl>p`           |        Find file         |
